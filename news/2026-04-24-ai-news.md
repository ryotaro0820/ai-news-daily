### 1. DeepSeek、V4シリーズを電撃リリース——オープンソース最強モデルが100万トークンの文脈処理を実現

中国のAIスタートアップ DeepSeek は4月24日、最新フラッグシップモデル「**DeepSeek V4**」シリーズのプレビュー版をリリースした。上位モデルの **V4-Pro**（総パラメータ数1.6兆、1トークンあたり490億が活性化する MoE 構造）と、軽量・高速な **V4-Flash**（総パラメータ数2,840億）の2種類が提供される。

最大の特徴は、**100万トークン（約75万語）のコンテキストウィンドウ**をデフォルトで備えること。同社が「ハイブリッドアテンション・アーキテクチャ」と呼ぶ独自技術により、長い会話や大規模コードベース全体を1つのプロンプトとして処理できる。数学・コーディングのベンチマークではオープンソースモデルで最高スコアを記録し、Google の閉源モデル Gemini 3.1-Pro に次ぐ性能とされる。

**価格破壊**も話題を呼んでいる。V4-Pro の出力コストは 100万トークンあたり **$3.48** で、OpenAI ($30) の約9分の1、Anthropic ($25) の約7分の1という超低価格。両モデルは Hugging Face で公開され、ウェイトのダウンロードやローカル実行・ファインチューニングが可能だ。また、TencentとAlibabaがDeepSeekに **200億ドル超** の評価額で出資交渉中との報道もある。

### 2. OpenAI、エージェントモデル「GPT-5.5」を正式リリース——Terminal-Bench 2.0 で 82.7% の精度

OpenAI は4月23〜24日、社内コード名「Spud」として開発されてきた **GPT-5.5** および **GPT-5.5 Pro** をリリースした。従来モデルと異なり、複数のツールを自律的に使い分けながら複雑なタスクを完遂する「エージェント型」設計が最大の特徴だ。

コーディング・コンピューター操作・知識業務・初期科学研究の分野で特に高い改善が見られ、複雑なコマンドラインワークフローを評価するベンチマーク「**Terminal-Bench 2.0**」では **82.7%** の精度を達成し、現時点で最高性能を記録した。

GPT-5.5 は ChatGPT の Plus・Pro・Business・Enterprise プランおよび Codex で利用可能。API も同時に開放された。OpenAI の年間換算収益は **250億ドル** を突破しており、2026年後半にも IPO（株式公開）の準備を進めているとされる。

### 3. Google Cloud Next 2026：Vertex AI が「Gemini Enterprise Agent Platform」に——A2A プロトコル v1.2 が本番稼働

4月22日に開幕した **Google Cloud Next 2026** で、Google は企業向けエージェントAI基盤の全面刷新を発表した。Vertex AI は「**Gemini Enterprise Agent Platform**」として再ブランド化され、200種類以上のモデル（Anthropic の Claude を含むサードパーティ製も含む）を選択できるプラットフォームへと進化した。

エージェント間の通信規格 **Agent2Agent（A2A）プロトコル** はバージョン 1.2 へ更新され、**Linux Foundation** 傘下の「Agentic AI Foundation」に移管。Microsoft・AWS・Salesforce・SAP・ServiceNow を含む **150組織** が既に本番環境で運用中だ。A2A は Anthropic の Model Context Protocol（MCP）を補完し、異なるプラットフォーム間でエージェントを連携させる役割を担う。

その他の発表として、Google Workspace 向けのノーコード・エージェントビルダー、Webブラウジング AIエージェント「**Project Mariner**」、Wiz との統合によるセキュリティ強化も行われた。

- 出典: [The Next Web](https://thenextweb.com/news/google-cloud-next-ai-agents-agentic-era) / [Google Developers Blog](https://developers.googleblog.com/en/a2a-a-new-era-of-agent-interoperability/) / [SiliconANGLE](https://siliconangle.com/2026/04/22/google-cloud-next-new-security-operations-agents-wiz-integrations-agent-governance-tools/)

### 4. AI脅威の加速——フロンティアモデルの能力、4か月ごとに倍増と英国AISI報告

英国の AI 安全機関（AISI）は最新レポートで、フロンティアモデルの能力が **以前の8か月サイクルから4か月サイクル** へと倍速で強化されていると評価した。特にサイバーセキュリティへの影響が深刻で、AIが「ソフトウェアの脆弱性の発見→悪用コード作成→大規模実行」という工程を、以前は希少な専門家のみが担えた速度・スケールで実現できるようになっていると警告している。

英国政府はこれに合わせてビジネスリーダー向けの公開書簡を発表。AIによるサイバー脅威に対する組織的な準備を呼びかけた。

## 📄 注目の研究・論文

### ICLR 2026 開幕（4月23〜27日、リオデジャネイロ）——信頼性・効率・安全のAI研究を集中展示

**ICLR 2026**（第14回学習表現国際会議）が4月23日、ブラジルのリオデジャネイロで開幕した。Salesforce AI Research は21本の採択論文を発表。Lambda も12本の論文と2つのワークショップに参加しており、**長期的なエージェントプランニング**・**推論時スケーリング**・**マルチモーダル効率化** などが主要テーマとなっている。

注目の研究：
- **ECF8（Exponent-Concentrated FP8）**: Huffman符号化を利用した新量子化方式。671B規模のLLMで完全無損失、拡散モデルのメモリ使用量を **最大26.9%削減**、スループットを **最大177%向上**。
- **再帰言語モデル（RLM）**: 任意長のプロンプトを外部環境として段階的に処理する推論時スケーリング手法。超長文脈処理の新アプローチとして注目。
- AIブレークスルーがエネルギー効率を **100倍改善** しつつ精度も向上させたとの研究も報告されている。

- 出典: [Salesforce AI at ICLR 2026](https://www.salesforce.com/blog/salesforce-iclr-2026/) / [Lambda ICLR 2026](https://lambda.ai/blog/iclr-2026-12-papers) / [ScienceDaily](https://www.sciencedaily.com/releases/2026/04/260405003952.htm)

## 📊 業界全体トレンド

- **AI経済格差の拡大（PwC調査）**: AIによる経済的利益の75%を上位20%の企業が独占。生産性向上だけでなく「収益拡大」を優先する企業群が急速にリードを広げている。
- **オープンソース vs クローズドの競争激化**: DeepSeek V4-Pro が最強オープンソースモデルとなり、OpenAI や Anthropic の閉源モデルと真っ向勝負。価格差（約8〜9倍）が市場を揺るがす。
- **エージェントAIが主戦場へ**: OpenAI GPT-5.5、Google の Gemini Enterprise Agent Platform、A2A プロトコルと、各社の戦略がエージェント型AIへ集中。2026年はエージェント元年となる様相。
- **AIサイバー脅威の臨界点**: 英国AISIの報告は、今後1〜2年でAIが国家レベルのサイバー攻撃を自律実行できる水準に達するリスクを示唆している。

## 📝 今日のまとめ

2026年4月24日は、AIの歴史において特筆すべき一日となった。**DeepSeek V4** は1年前に世界を震撼させた DeepSeek R1 の再来とも言える衝撃をもたらし、超低価格・高性能・オープンソースというトリプルパンチで市場に波紋を広げた。同日に **OpenAI GPT-5.5** もリリースされ、エージェントAIの覇権を巡る競争がいよいよ本格化した。

Google の A2A プロトコル標準化と Anthropic の収益優位は、AIが「研究から産業インフラへ」移行した現実を象徴している。一方でフロンティアモデルの能力加速（4か月で倍増）は、英国政府が公開書簡で企業に警告を発するほど深刻な安全課題となっている。技術の進歩と安全・経済格差の課題をどう両立するかが、業界全体の焦点となっている。
