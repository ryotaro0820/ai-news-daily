## 🔥 主要ニュース

### 1. Anthropic、「Claude Opus 4.7」を正式リリース——初の高解像度ビジョン対応

Anthropic は4月16日、最新フラッグシップモデル **Claude Opus 4.7** を正式にリリースした。前バージョン（Opus 4.6）に比べ、エージェント型コーディングベンチマーク（93タスク）の解決率が **13%向上**。また、本モデルは初めて高解像度画像入力（最大 2576px / 3.75MP）に対応した。

開発者向けの新機能として「**タスクバジェット**」を導入。Claude がエージェントループ全体で使うべきトークン量の目安を事前に設定でき、モデルが残り時間を意識しながらタスクを効率的に完了できるようになった。さらに、推論の深さを調整する新設定値 `xhigh` が追加され、`high` と `max` の間できめ細かいコントロールが可能に。価格は Opus 4.6 と同一（入力100万トークンあたり5ドル、出力25ドル）で据え置き。

なお、同社のより高性能なモデル **Claude Mythos Preview** は引き続き Project Glasswing を通じた限定公開のみで、一般提供は行われていない。

- 出典: [Anthropic公式](https://www.anthropic.com/news/claude-opus-4-7) / [GitHub Changelog](https://github.blog/changelog/2026-04-16-claude-opus-4-7-is-generally-available/) / [AWS](https://aws.amazon.com/about-aws/whats-new/2026/04/claude-opus-4.7-amazon-bedrock/)

### 2. AIコーディングスタートアップ Cursor、評価額500億ドル超で20億ドル調達交渉中

AIコーディングエディタの最大手「**Cursor**」（Anysphere社）が、**20億ドル（約3,000億円）** の新規資金調達に向けた交渉を進めていることが4月19日に報じられた。調達後の企業評価額は **500億ドル超** となり、昨年11月の293億ドルから約1.7倍に急騰する見込みだ。

今回のラウンドはベンチャーキャピタル大手の **Andreessen Horowitz（a16z）** が共同リードを担い、**NVIDIA** および **Thrive Capital** も参加予定。Cursorの年間換算収益（ARR）は2025年2月に20億ドルを突破し、2026年末には **60億ドル超** を見込む。開発者チームの80%がAIツールを積極活用し、コードの受け入れ率が20%から60%に急上昇していることがこの急成長を支えている。

- 出典: [CNBC](https://www.cnbc.com/2026/04/19/cursor-ai-2-billion-funding-round.html) / [TechCrunch](https://techcrunch.com/2026/04/17/sources-cursor-in-talks-to-raise-2b-at-50b-valuation-as-enterprise-growth-surges/) / [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-17/ai-coding-startup-cursor-in-talks-to-raise-2-billion-in-funding)

### 3. EU、AI採用ツールのバイアス審査規制を確定——105日後に施行

EU AI法の施行期限（8月2日）まで残り **105日** となった4月19日、欧州当局は AI採用ツールに対する監査スコープ・文書要件・認証基準を正式に確定した。

対象となる企業は、トレーニングデータの構成情報、性別・年齢・民族などの属性別の採否データ、継続的モニタリングの証拠を準備する義務を負う。コンプライアンス違反に対するペナルティは **€1,500万または全世界年間売上の3%のいずれか高い方**。

認証済み第三者監査人の数は限られており、すでに予約が埋まり始めているとされる。8月2日に向けて今すぐ監査契約を締結する必要があると専門家は警告している。

- 出典: [Asanify AI News Digest](https://asanify.com/blog/news/ai-hiring-bias-audit-april-19-2026/)

### 4. OpenAI・Anthropic・Google が中国によるモデル盗用に連携対抗——「Frontier Model Forum」で情報共有

4月6〜7日にかけて、**OpenAI・Anthropic・Google** の3大AI企業は「**Frontier Model Forum**」を通じてインテリジェンスを共有し、中国企業による「敵対的蒸留（Adversarial Distillation）」への対策を強化すると発表した。

Anthropicによれば、**DeepSeek・Moonshot AI・MiniMax** の3社が約24,000の不正アカウントを通じて **1,600万件以上** のClaudeとのやり取りを行い、同社モデルの能力を無断で複製しようとしていたと指摘。欧米最大手のAI企業が協調してモデル保護に乗り出す前例のない連携として注目される。

- 出典: [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-06/openai-anthropic-google-unite-to-combat-model-copying-in-china) / [The Japan Times](https://www.japantimes.co.jp/business/2026/04/07/tech/openai-anthropic-google-china-copy/)

### 5. Microsoft、Windows 11 タスクバーに AI エージェントを統合

Microsoftは今週より、**Windows 11 のタスクバーへの AI エージェント統合**の展開を開始した。ユーザーはデスクトップ環境から直接AIエージェントにアクセスでき、アプリ間をまたいだタスク処理や情報取得が可能になる。OSレベルでのAI統合が本格化し、パーソナルコンピュータの操作体験が大きく変わる転換点となりそうだ。

- 出典: [Daily AI Agent News](https://aiagentstore.ai/ai-agent-news/this-week)

### 6. 企業の86%がAIエージェントのアクセスポリシーを未整備——セキュリティ懸念が深刻化

最高情報セキュリティ責任者（CISO）への調査によると、**86%の組織がAIエージェントに対するアクセスポリシーを施行していない**ことが判明した。さらに、AIエージェントが侵害された場合に「封じ込めできる」と回答した割合はわずか **5%** にとどまった。

AIエージェントは多くの場面で管理者レベルのアクセス権を持ちながら、ほとんど監視されていない状態で動作している。企業のAI活用が急拡大する一方、ガバナンスとセキュリティへの対応が追いついていない実態が浮き彫りになった。

- 出典: [AI Agent Store — Daily AI Agent News](https://aiagentstore.ai/ai-agent-news/this-week)

### 7. スタンフォードAI Index 2026：中国が米国のAIリードを「ほぼ消滅」させつつある

スタンフォード大学が4月16日に公開した **AI Index 2026** によると、中国は急速なキャッチアップにより米国とのAI性能格差を大幅に縮小しており、一部のベンチマークでは米国に **「ほぼ並んだ」** 状態にあるという。同レポートは、米国への技術人材の流入が減速していることも懸念材料として挙げている。

また同レポートは、AI性能の成長速度が PC やインターネットの普及期を超えており、AI企業の収益成長は過去のあらゆる技術革命を上回る水準と報告。一方で、AIへの巨大な設備投資（データセンター、半導体）による長期リスクも指摘している。

- 出典: [Fortune — Stanford Study](https://fortune.com/2026/04/16/stanford-study-how-has-china-gained-on-us-ai-war/) / [IEEE Spectrum](https://spectrum.ieee.org/state-of-ai-index-2026)

## 📄 注目の研究・論文

### ICLR 2026 が4月23〜27日にリオデジャネイロで開幕——採択論文5,300件超

機械学習分野の最重要会議のひとつ、**ICLR（International Conference on Learning Representations）2026** が4月23〜27日、ブラジルのリオデジャネイロで開催される。今回の採択論文数は **5,300件超** と過去最多規模。Apple・Google・Anthropicなどの主要企業も独自研究を発表予定で、長文コンテキスト処理の推論時スケーリング、グラフベースメモリを持つ LLM アーキテクチャ、金融センチメント分析などが主要トピックとなっている。

- 出典: [ICLR 2026 公式](https://iclr.cc/) / [Apple Machine Learning Research](https://machinelearning.apple.com/updates/apple-at-iclr-2026)

### ACL 2026 採録：LLMのジェイルブレイクに関する新論文

KAIST の Statistical AI Lab が「**Incomplete Prompt Jailbreaks in Large Language Models**」を ACL 2026 Findings に採録されたと発表。不完全なプロンプトを用いてLLMの安全ガードを回避する手法を分析した研究で、モデルの安全性評価・強化に貢献するとされる。
