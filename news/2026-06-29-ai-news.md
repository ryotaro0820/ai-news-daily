### 3. Anthropic、Alibaba Qwen ラボによる「2880万件の蒸留攻撃」を正式告発

Anthropic が米国上院銀行委員会に宛てた書簡（6月10日付）で、**Alibaba の Qwen AI ラボに関連するオペレーター**が Claude モデルへの組織的な**蒸留攻撃（distillation attack）**を実施したと公式に告発したことが明らかになっています。

**被害の概要：**
- 期間：2026年4月22日〜6月5日
- 不正アカウント数：**約2万5,000件**
- 生成されたやり取り：**2880万件以上**
- 目的：Claude の推論・プログラミング・複雑タスク実行能力の分析・模倣

蒸留攻撃とは、大量のクエリを通じてターゲットモデルの能力を抽出し、自社モデルの学習データとして活用する手法です。Anthropic はこれを **知的財産の窃取** と位置づけており、法的措置の行方が注目されています。

### 4. Figma、Config 2026 で AI アニメーションツール「Motion」をオープンベータ公開

6月24日から段階的に展開が始まった **Figma Motion** は、デザインファイル内でプロダクションレディなアニメーションを作成できる新機能です。

**主な機能：**
- ネイティブのアニメーションタイムライン・キーフレーム編集
- **AI エージェントによる自動キーフレーム生成**（「こんな動きにして」と指示するだけ）
- プリセットアニメーションとイージング曲線・スプリングアニメーション
- Dev Mode 対応・CSS / JSON / React / MP4 / Animated SVG / GIF など多形式エクスポート
- デザインシステムへの組み込み（コンポーネントにモーションを付与すれば全ファイルに反映）

全プランで**オープンベータとして無料利用可能**。Figma はモーションデザインをデザインシステムの基盤コンポーネントに位置づけており、After Effects や Lottie を使う従来のワークフローを大きく変える可能性があります。

## 注目の研究・論文

### Anthropic、Claude Code 40万セッション分析 — 「専門知識 > プログラミング能力」

Anthropic が **2025年10月〜2026年4月**の Claude Code セッション約**40万件**を分析した研究を発表しました。

**主な発見：**
- Claude が自律的に実行する作業量を決定するのは、ユーザーの**プログラミング能力ではなくドメインの専門知識**
- 医療・法律・金融など専門領域のユーザーは、コーディングスキルが低くても Claude への委任度が高い
- 逆に熟練エンジニアでも「慣れない領域」では監督的な関与を保つ傾向がある

この知見は、AI ツールの設計において**技術リテラシーより専門性の架け橋が重要**であることを示唆しており、エンタープライズ展開の戦略にも影響を与えそうです。

### Agentjacking — AI コーディングエージェントを標的とした新たな攻撃手法

6月13日、Tenet Security が **「Agentjacking」** と名付けた攻撃手法を公開しました。偽の Sentry エラーレポートにマークダウンインジェクションを仕込み、AI コーディングエージェントを乗っ取る手法で、**2,388 組織**が影響を受けうる脆弱性が確認されています。

- 実証実験での**エクスプロイト成功率：85%**
- 標的：Claude Code・Copilot・Cursor などのエージェント型 IDE 拡張

AI コーディングツールの普及とともに、AI エージェントを狙ったサイバー攻撃が現実の脅威となっています。
