## 主要ニュース

### 1. Anthropic、Blackstone・Goldman Sachsと総額15億ドルのエンタープライズAI合弁会社を設立

Anthropicは5月4日、大手プライベートエクイティ（PE）企業のBlackstone、Hellman & Friedman、Goldman Sachsと連携し、企業向けAIサービスを提供する新会社を設立すると発表した。資本規模は **約15億ドル（約2250億円）**。

新会社はPEファンドが保有する中堅企業へClaude AIを展開し、導入支援から運用最適化まで一貫して担う。Anthropicはカスタムソリューション構築と長期サポートのためのエンジニアを派遣し、AI導入を単なる試験的利用から「コアオペレーション」への定着へと引き上げることを目指す。Blackstone・Hellman & Friedman・Anthropicがそれぞれ約3億ドル、Goldman Sachsが1億5000万ドルを出資。General Atlantic・Apollo Global Management・GIC・Sequoia Capitalなども参加する。

エンタープライズAI市場でOpenAIとの競争が激化するなか、AnthropicはPE資本を活用して企業導入を一気に加速させる戦略だ。

### 2. OpenAIも同日、「The Deployment Company」を総額100億ドル規模で設立

OpenAIは5月4日、企業へのAI展開に特化した合弁会社 **「The Deployment Company」** を設立すると発表した。19社の投資家から **40億ドル超** を調達し、会社全体の評価額は **約100億ドル（約1.5兆円）** に達する。

出資者にはTPG、SoftBank Group、Bain Capital、Brookfield Asset Management、Dragoneer Investment Groupなどが名を連ねる。OpenAI自身は5億ドルを拠出（最大15億ドルまで追加可能）し、過半数の経営権を維持する。PE投資家には年率17.5%の保証リターンが設定されている。

Anthropicの15億ドル合弁と同日発表となり、エンタープライズAI展開を巡るAnthropicとOpenAIの競争が企業規模でも激突する形となった。

### 3. 米国防総省、8社と機密AIネットワーク協定 — Anthropicは除外

米国防総省（Pentagon）は5月1日、Microsoft・Amazon Web Services・Google・OpenAI・NVIDIA・SpaceX・Oracle・Reflectionの計8社と、軍の機密ネットワーク（IL6/IL7）へのAI展開に関する協定を締結したと発表した。

AIは戦場でのデータ分析・状況認識・意思決定支援に活用される予定。一方、**Anthropicは協定から除外** された。同社が「完全自律型兵器への利用禁止」「大規模な国内監視への利用禁止」といった安全ガードレール条項の明記を求めたことがトランプ政権との摩擦となり、国防総省はAnthropicを「安全保障上のリスク」として事実上の締め出しにした形だ。

今回の動きはAI軍事利用における「安全性」と「国家安全保障」の価値観の衝突を鮮明に示している。

### 4. Google Cloud Next 2026 — Vertex AIを「Gemini Enterprise Agent Platform」に刷新

Googleは先週開催した **Google Cloud Next 2026** で、Vertex AIを新プラットフォーム **「Gemini Enterprise Agent Platform」** へと再編・強化すると発表した。

新プラットフォームには Agent Studio（ビジュアルフロービルダー）、Agent-to-Agent Orchestration（エージェント間連携）、Agent Registry（管理レジストリ）、Agent Identity（ID管理）、Agent Gateway、Agent Observability（監視機能）などが統合される。また、**Anthropic Claude Opus 4.7** を含む200以上のモデルをサポートするマルチモデル環境を提供。**Agentic Data Cloud** も発表され、AIエージェントが大規模データを高速活用できる新アーキテクチャを導入する。

**ソース:** [Google Cloud Blog](https://cloud.google.com/blog/products/ai-machine-learning/introducing-gemini-enterprise-agent-platform) / [Virtualization Review](https://virtualizationreview.com/articles/2026/04/24/google-cloud-next-26-gemini-enterprise-agent-platform-leads-ai-centric-news.aspx)

## 注目の研究・論文

### Google「TurboQuant」— KVキャッシュのメモリ課題を解決する新アルゴリズム（ICLR 2026）

Googleの研究チームが **TurboQuant** を発表。PolarQuantベクトル回転と量子化Johnson-Lindenstrauss圧縮の2段階処理により、大規模言語モデルの推論における最大のボトルネックの一つであるKVキャッシュのメモリオーバーヘッドを大幅に削減する。長いコンテキストウィンドウを持つモデルをより効率的に動作させ、オンデバイスAIやデータセンターコストの削減に貢献する技術として注目される。

**ソース:** [LLM Stats AI News](https://llm-stats.com/ai-news)

### MIT — LLM学習速度を2倍化する新手法を発表

MITの研究者が、推論モデルのトレーニング効率を大幅に向上させる新手法を発表した。計算の「空き時間」を活用し、大規模推論LLMの出力を予測する小型・高速モデルを自動的に並行訓練することで、**精度を維持しながらトレーニング速度を2倍** に高めることに成功した。

**ソース:** [MIT News](https://news.mit.edu/2026/new-method-could-increase-llm-training-efficiency-0226)
