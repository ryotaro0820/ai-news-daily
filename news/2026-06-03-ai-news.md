## 主要ニュース

### 1. Microsoft Build 2026 Day 2：MAIモデル7本投入、「エージェントファースト」時代宣言

Microsoft Build 2026の2日目（6月2〜3日）に、Mustafa Suleiman氏が旗艦推論モデル **MAI-Thinking-1**（350億パラメータ・コンテキスト256K）を発表。さらに、文章からコードを自動生成するコーディング専用モデル **MAI-Code-1-Flash** をGitHub CopilotおよびVS Codeに統合した。

今回発表されたMAIシリーズは計7モデルに及ぶ。

| モデル名 | 特徴 |
|---------|------|
| MAI-Thinking-1 | 旗艦推論モデル（35B、256Kコンテキスト） |
| MAI-Code-1-Flash | GitHub/VS Code向けコード生成 |
| MAI-Image-2.5 / Flash | テキスト→画像・画像→画像両対応（初） |
| MAI Transcribe 1.5 | 43言語対応・高精度音声認識 |
| MAI-Voice-2 / Flash | 15以上の言語で新音声オプション |

CEO Satya Nadella氏は「オプショナリティ」という言葉を繰り返し強調し、OpenAIへの依存を意図的に減らす戦略転換を示唆した。GitHub Copilotのデスクトップアプリもプレビュー公開された。

### 2. Anthropic、評価額9,650億ドルでIPO申請——OpenAIを初めて上回る

Anthropicは6月2日、SEC（米証券取引委員会）へのIPO草案（S-1）をコンフィデンシャル提出したと発表。直前に完結した調達ラウンドはポストマネー評価額 **9,650億ドル**（約145兆円）に達し、OpenAIの8,520億ドルを上回り、**世界で最も高い評価額を持つ非上場AI企業**となった。

先月参画したAndrej Karpathy氏（元OpenAI・元Tesla AI責任者）が事前学習チームを率いることも改めて確認されており、技術人材獲得でも業界をリードしている。

- [Anthropic IPO申請（TechCrunch）](https://techcrunch.com/2026/06/01/anthropic-files-to-go-public/)
- [AnthropicのS-1提出発表](https://www.anthropic.com/news/confidential-draft-s1-sec)

### 3. Gemini 3.5 Flash がGA——フロンティア級性能を4倍速・低コストで

Google の **Gemini 3.5 Flash** が正式GA（Generally Available）となった。同等クラスのモデル比で4倍高速、価格は入力$1.50／出力$9.00（100万トークンあたり）。コンテキストウィンドウは100万トークン。すでにGeminiアプリおよびGoogle検索の「AI Mode」でデフォルトモデルとして採用されている。

Gemini 3.5 ProはGoogleのSundar Pichai CEOが「6月中に提供」と明言しており、近日公開予定だ。

- [June 2026 AI Launch Wave（WaveSpeed Blog）](https://wavespeed.ai/blog/posts/june-2026-ai-launch-wave/)

### 4. GitHub Copilot、トークン課金制に完全移行

6月1日より、GitHub Copilotがすべてのプランでトークンベース課金へ切り替わった。従来のフラット月額サブスクリプションから移行することで、軽度ユーザーはコスト削減、ヘビーユーザーは使用量に応じた支払いが求められる。Microsoft Build での GitHub Copilot アプリ（デスクトップ・プレビュー）発表とあわせ、開発者向けAIエコシステムの再編が加速している。

- [AI News June 3, 2026（Build Fast With AI）](https://www.buildfastwithai.com/blogs/ai-news-today-june-3-2026)

### 5. 米国防総省、Microsoft に96.9億ドルの大型AI契約

米国防総省（DoD）がMicrosoft 365・Azure・AI Copilotサービスを一括導入する約 **96.9億ドル（約1.4兆円）** の大型エンタープライズ契約を締結した。分散したベンダー契約を統合することで 4.22億ドルのコスト削減を見込む。軍・政府機関へのAI普及が大規模に進む節目となった。

## 注目の研究・論文

2026年のLLM研究は「効率・安全・長文脈推論・マルチモーダル」の4テーマに集中している。最近注目されている論文：

- **AdapTime**：時間的複雑さに応じて推論アクション（再定式化・書き直し・レビュー）を動的に選択する手法
- **Tool-DC**：divide-and-conquerフレームワークでツール選択の「試行→確認→再試行」を改善
- **Latent World Models**：潜在ベクトルで環境状態を圧縮するワールドモデルを言語モデルと共同学習し、長期タスクの推論コストを約1/3削減
- **Exploratory Sampling**：表層変化ではなく意味的多様性を促すデコード手法（軽量テスト時蒸留器を活用）
