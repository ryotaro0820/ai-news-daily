### 1. Anthropic、Karpathy氏を迎えてClaude事前学習研究チームを正式始動

### 2. Claude Opus 4.8 リリース＆Dynamic Workflows 研究プレビュー公開

Anthropicは **Claude Opus 4.8** をリリース。Opus 4.7 に比べて各種ベンチマークで改善されており、同価格で提供される。あわせて **Dynamic Workflows** が Claude Code CLI・Desktop・VS Code 拡張機能で研究プレビュー公開された。Dynamic Workflows は大規模タスクを複数エージェントが並列・階層的に処理する機能で、Max・Team・Enterprise プランおよび Claude API・Amazon Bedrock・Vertex AI・Microsoft Foundry でも利用可能。さらに Fast Mode（2.5倍速）は旧モデル比で**3倍安価**になった。

加えて、法律事務所・企業法務向けに **20以上の新法律 MCP コネクタ** と12の実務領域プラグインが公開され、リサーチ・契約・eDiscovery など幅広い法務ワークフローに対応する。

### 4. Google I/O 2026 ふりかえり — Gemini 3.5 Flash・Omni・Spark が本格展開へ

5月19日に開催された **Google I/O 2026** で発表された3モデルが本格展開フェーズに入っている。

| モデル | 特徴 |
|--------|------|
| **Gemini 3.5 Flash** | Gemini 3.1 Pro をコーディング・エージェント・マルチモーダル全ベンチマークで上回り、出力速度は競合フロンティアモデルの**4倍**、コストは**半額以下** |
| **Gemini Omni** | 画像・音声・動画・テキストを入力として高品質動画を生成する"ワールドモデル" |
| **Gemini Spark** | 接続アプリ間を横断推論し24時間365日稼働するパーソナル AI エージェント。Google AI Ultra 加入者向けにベータ提供開始 |

また Google Cloud では Vertex AI を **Gemini Enterprise Agent Platform** に改称し、Workspace Studio（ノーコードエージェント構築）や Model Garden（Anthropic Claude 含む 200以上のモデル）を統合した。

## 注目の研究・論文

### 安全性・エージェント・時間推論の3テーマが最前線

2026年5月末のLLM研究動向として、以下のテーマが主要会議（ACL 2026 等）に向けて注目されている。

- **Tool-DC**：多数のツール候補がある長コンテキスト環境でのツール呼び出し精度を、分割統治（Divide and Conquer）フレームワークで改善する手法。エージェントAIの実用化を加速する。
- **AdapTime**：LLMの弱点とされる時間的推論（Temporal Reasoning）を動的に改善するフレームワーク。問いの複雑さに応じて推論戦略を自動切り替えする。
- **数学的発見支援**：数学者が長期にわたる研究を継続できるステートフル AI ワークスペースの研究が進行中。
- **Google DeepMind の安全性論文**：LLMが人間の信念や行動に影響を与える説得的な出力を生成し得るかを分析した論文が公開。
