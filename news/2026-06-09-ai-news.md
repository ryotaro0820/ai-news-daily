### 1. OpenAI が IPO 申請書を極秘提出 ― Anthropic の1週間後を追う形で
OpenAI は2026年6月8日、米SEC（証券取引委員会）に対して S-1 登録届出書を極秘提出し、株式上場に向けた手続きを開始した。Anthropic が6月1日に同様の届出を行ってからわずか1週間後のことで、業界最大手2社が相次いでIPOレースに突入した格好だ。OpenAI の現在の民間市場評価額は約 **8,520億ドル**。Anthropic はシリーズHで **650億ドル**を調達し評価額 **9,650億ドル** とOpenAIを上回っており、史上最も価値の高い未上場AI企業となっている。

### 2. Microsoft、Build 2026 で自社開発 MAI モデル 7本を発表
Microsoft は6月2日の Build 2026 にて、OpenAI に依存しない独自開発モデルファミリー「**MAI**」を一挙7本公開した。主なモデルは以下の通り。

| モデル | 概要 |
|---|---|
| **MAI-Thinking-1** | 35Bパラメータ MoE・256Kコンテキスト。ソフトウェアエンジニアリングベンチで Sonnet 4.6 に競合 |
| **MAI-Code-1-Flash** | 5Bパラメータ、GitHub Copilot 深統合の軽量コーディングモデル |
| **MAI-Image-2.5** | テキスト→画像生成＋画像編集対応、Arena スコアでトップクラス |
| **MAI-Transcribe-1.5** | 世界最高精度と主張する音声認識モデル |
| **MAI-Voice-2** | 15言語対応の自然音声合成モデル |

同社は「OpenAI を含む第三者モデルからのディスティレーションなし」でゼロから学習したと強調。独立した AI 基盤構築への本格転換を示した。

- 出典：[Microsoft AI](https://microsoft.ai/news/building-a-hillclimbing-machine-launching-seven-new-mai-models/) / [CNBC](https://www.cnbc.com/2026/06/02/microsoft-unveils-new-ai-models-lessen-reliance-on-openai-lower-costs.html)

### 3. Anthropic の ARR が 300億ドル突破・Claude Code 単体で 25億ドル
Anthropic の年換算収益（ARR）が4月時点で **300億ドル** を超え、OpenAI（約250億ドル）を逆転したと報じられている。CEO ダリオ・アモデイ氏は「自社予測の8倍以上の成長速度」と語った。

特筆すべきは、コーディング支援ツール **Claude Code** の急成長。2025年5月に一般公開されてから約9ヶ月で ARR **25億ドル** を達成し、エンタープライズ向けAnthropicの売上の過半数を占める。B2Bソフトウェアの歴史上、これほど急速な成長は前例がないとされる。

### 4. 世界初：LLM エージェントが自律的にサイバー攻撃を実行、60分以内に AWS データベースを侵害
セキュリティ企業 **Sysdig** の脅威調査チームが、LLM エージェントを用いた実際の攻撃を世界で初めて観測・記録した。

攻撃の流れは以下の通り：
1. Python ノートブックプラットフォーム **Marimo** の重大な RCE 脆弱性（CVE-2026-39987）を悪用
2. 環境変数・設定ファイルから AWS 認証情報を抽出
3. AWS Secrets Manager API から SSH 秘密鍵を取得
4. 内部 PostgreSQL データベースのスキーマと全データを **2分以内**に流出

人間のオペレーターが介入することなく、AI エージェントがリアルタイムに判断・適応しながら全ポストエクスプロイトチェーンを完遂した。フォレンジック調査では「中国語のコメント」「機械最適化コマンド形式」など4つの行動的特徴が確認された。

- 出典：[Sysdig Blog](https://www.sysdig.com/blog/ai-agent-at-the-wheel-how-an-attacker-used-llms-to-move-from-a-cve-to-an-internal-database-in-4-pivots) / [Security Magazine](https://www.securitymagazine.com/articles/102325-ai-agent-conducted-a-cyberattack-on-its-own-it-took-less-than-one-hour)

## 注目の研究・動向

- **プロンプト内の情報位置が回答を左右**：50のフロンティアモデルを対象にした3年間の研究で、「プロンプトのどこに情報を置くか」が「何を書くか」よりも回答に影響することが判明。RAG設計やシステムプロンプト構築に大きな示唆を与える。
- **OpenAI が "完全自律AI研究" を2028年目標から撤退**：「人間とAIのタンデム」体制へ方針転換。完全な自律研究ではなく、協調型アプローチを強調する方向に。

## 今日のまとめ

2026年6月9日時点で、AI業界は **IPOラッシュ**・**大手独自モデル競争**・**セキュリティリスクの現実化** という三重の転換点を迎えている。AnthropicとOpenAIの相次ぐ上場申請は、数兆円規模の資本調達と公開市場への参入を意味し、業界構造を一変させる可能性がある。Microsoft の MAI モデル群はOpenAI依存からの自立を鮮明にし、各社の垂直統合が加速。一方でSysdigが記録した初の自律型LLMサイバー攻撃は、「AIが道具から攻撃主体へ」というリスクが現実のものとなったことを示している。LLMはもはや実験ツールではなく、ビジネスの基幹インフラかつセキュリティ上の脅威として機能する段階に入った。
