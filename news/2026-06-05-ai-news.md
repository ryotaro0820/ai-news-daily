## 主要ニュース

### 1. OpenAI、ChatGPTに「Dreaming V3」メモリー機構を導入——過去の会話を自律的に学習

OpenAIは6月4〜5日にかけて、ChatGPT Plus・Proユーザー（米国内）向けに**Dreaming V3**と呼ばれる新しいメモリーアーキテクチャの展開を開始した。

従来の「保存済みメモリーリスト」を手動で管理する方式から、**バックグラウンドで過去の会話全体を自動的に合成・更新する方式**へと根本的に刷新された。具体的には、「7月にシンガポールに行く予定」というメモリーが旅行後に自動で「2026年7月にシンガポールへ行った」へ書き換わるなど、ユーザー操作なしに文脈を最新化する。

一方でプライバシー面の懸念も浮上している。新アーキテクチャでは**メモリーがどのように合成されたかの監査証跡（audit trail）が削除**されており、ユーザーが「なぜAIがこれを知っているのか」を把握しにくくなるという指摘がある。Free・Go・国際ユーザーへの展開は今後数週間内に予定。

- [OpenAI upgrades ChatGPT memory with Dreaming（gHacks）](https://www.ghacks.net/2026/06/05/openai-upgrades-chatgpt-memory-with-new-dreaming-architecture-for-plus-and-pro-users/)
- [ChatGPT Dreaming V3 Memory Update（TechTimes）](https://www.techtimes.com/articles/317840/20260605/chatgpt-memory-dreaming-update-openai-rewrites-personalization-engine-limits-audit-trail.htm)

### 2. Microsoft Build 2026——自社開発7モデル投入、MAI-Thinking-1がOpenAIとの距離を置く

Microsoftは6月2〜3日に開催した**Build 2026**において、自社開発AIモデル7本を一挙公開した。目玉は**MAI-Thinking-1**——OpenAIのデータを一切使わずに商用ライセンスデータのみで訓練した同社初のフルインハウス推論モデルだ。

MAI-Thinking-1の主要スペックは以下のとおり：

| 項目 | 詳細 |
|------|------|
| アーキテクチャ | スパース Mixture-of-Experts（MoE） |
| アクティブパラメーター | 350億（総パラメーターは約1兆） |
| コンテキストウィンドウ | 256,000トークン（約600ページの文書を一括処理可能） |
| 学習データ | OpenAIデータなし、商用ライセンスデータのみ |

さらにMicrosoftは**Scout**（Microsoft 365向け常時稼働AIアシスタント）、Copilot Studioのドラッグ&ドロップ型エージェントワークフローキャンバス、超低遅延の音声エージェントなども発表した。

- [Microsoft MAI-Thinking-1 at Build 2026（TechTimes）](http://www.techtimes.com/articles/317631/20260602/microsoft-build-2026-mai-thinking-1-first-house-reasoning-model-trained-without-openai-data.htm)
- [Microsoft launches seven in-house AI models（Windows Central）](https://www.windowscentral.com/software-apps/microsoft-launches-seven-in-house-ai-models-to-cut-developer-costs-and-reduce-reliance-on-openai)

### 3. Google、Gemini 2.0 API を完全停止——3.5 Flashへの強制移行完了

Googleは6月1日（月）をもって**Gemini 2.0 API全体を停止**した。以降のAPIコールはすべてGemini 3.5 FlashまたはGemini 3.1 Flash-Liteへ自動移行される。

5月19日のGoogle I/O 2026でGAとなったGemini 3.5 Flashは同等ベンチマークで3.1 Proを上回るうえ、速度は約4倍。価格は入力100万トークンあたり1.50ドル、出力9.00ドルで、コスト効率でも競合を凌ぐ。Googleはさらに強力な**Gemini 3.5 Pro**を「来月」（7月）投入すると宣言しており、次のサプライズに期待が高まる。
