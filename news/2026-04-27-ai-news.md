## 🔥 主要ニュース

### 1. Google、Anthropic に最大400億ドルを投資——評価額3,500億ドルに

4月24日、Google（Alphabet）はAnthropicへの大型投資を発表した。まず現金100億ドルを即時投資し、Anthropicがパフォーマンス目標を達成した場合にさらに300億ドルを追加投資する最大**400億ドル規模**の契約だ。これによりAnthropicの評価額は**3,500億ドル**に上昇した。

現金投資に加え、Google は2027年から稼働予定の**5ギガワット分のTPUコンピュート容量**をAnthropicに提供することを約束しており、業界では「インフラ面の戦略的重要性は資金と同等かそれ以上」との声もある。Anthropicの年間換算収益は2024年末の10億ドルから2025年末に90億ドル、2026年4月初頭には驚異の**300億ドル**に達しており、この急成長がGoogleの大型ベットを後押しした形だ。

- 出典: [TechCrunch](https://techcrunch.com/2026/04/24/google-to-invest-up-to-40b-in-anthropic-in-cash-and-compute/) / [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-24/google-plans-to-invest-up-to-40-billion-in-anthropic) / [CNBC](https://www.cnbc.com/2026/04/24/google-to-invest-up-to-40-billion-in-anthropic-as-search-giant-spreads-its-ai-bets.html)

### 2. DeepSeek V4 プレビュー公開——競技プログラミングで史上最高スコアを記録

中国のAIスタートアップDeepSeekが4月24日、最新フラッグシップモデル「**DeepSeek-V4**」のプレビュー版を公開した。V4-Pro と V4-Flash の2バリアントで構成され、いずれも**MITライセンス**のオープンソースで提供される。

V4-Proは総パラメータ数1.6兆（アクティブ49B）、V4-Flashは284B（アクティブ13B）のMoEアーキテクチャを採用。デフォルトで**100万トークンのコンテキストウィンドウ**を持ち、Thinking/Non-Thinkingのデュアルモードに対応する。特筆すべきはV4-Proの**Codeforces レーティング 3,206** で、GPT-5.4の3,168を上回り**あらゆるAIモデル中で史上最高**の競技プログラミングスコアを達成。また、1Mトークンコンテキスト設定において、前世代V3.2と比較してFLOPs を73%削減、KVキャッシュを90%削減するなど推論効率も大幅に改善している。

- 出典: [DeepSeek API Docs](https://api-docs.deepseek.com/news/news260424) / [TechCrunch](https://techcrunch.com/2026/04/24/deepseek-previews-new-ai-model-that-closes-the-gap-with-frontier-models/) / [MIT Technology Review](https://www.technologyreview.com/2026/04/24/1136422/why-deepseeks-v4-matters/) / [Hugging Face](https://huggingface.co/deepseek-ai/DeepSeek-V4-Pro)

### 3. OpenAI、GPT-5.5 をリリース——ハルシネーション60%削減

OpenAIは4月23日、新モデル「**GPT-5.5**」を公開した。SWE-bench Verified で**88.7%** のスコアを達成し、GPT-5.4と比較してハルシネーション（幻覚）を**60%削減**したと発表。企業向けおよび科学的タスクでの推論精度・速度が向上している。

なお「GPT-6」については、Polymarket では依然「4月30日までにリリース」の確率が70%超とされているが、本記事執筆時点（4月27日）では未公開のまま。次期 GPT-6 ではサム・アルトマンCEOが「最重要機能」と呼ぶ**長期メモリ**と大幅に強化されたエージェント能力が搭載される見込みだ。

- 出典: [felloai.com](https://felloai.com/all-we-know-about-chatgpt-6/) / [llm-stats.com](https://llm-stats.com/llm-updates)

### 4. Google Cloud Next 2026——「エージェント時代」の本格到来を宣言

ラスベガスで開催された Google Cloud Next 2026 において、Googleは企業向けAIエージェントプラットフォーム「**Gemini Enterprise Agent Platform**」を発表した。これまでの Vertex AI を吸収・リブランドしたもので、エージェントの構築・ガバナンス・最適化を一元管理できるエンドツーエンドの環境を提供する。

主な新機能として、ノーコードのエージェントビルダー「Workspace Studio」、バックグラウンドで自律動作する「長時間稼働エージェント（Long-running Agents）」、そして200以上のモデルを搭載した「Model Garden」（Anthropic Claude を含む）が発表された。インフラ面では**第8世代TPU**も登場し、学習特化の TPU 8t と推論特化の TPU 8i に分かれ、後者は前世代比でコストパフォーマンスが**80%向上**するとされる。

- 出典: [Google Cloud Blog](https://cloud.google.com/blog/topics/google-cloud-next/welcome-to-google-cloud-next26) / [The Next Web](https://thenextweb.com/news/google-cloud-next-ai-agents-agentic-era)

## 📄 注目の研究・論文

### AIのエネルギー消費を最大100分の1に削減する新手法

ScienceDaily が報じた研究によると、ニューラルネットワークと人間に似たシンボリック推論を組み合わせた新アーキテクチャにより、AIのエネルギー消費を**最大100倍削減**しながら精度を向上できる可能性が示された。急増するAIデータセンターの電力需要への対応策として注目されている。

- 出典: [ScienceDaily](https://www.sciencedaily.com/releases/2026/04/260405003952.htm)

### MIT：推論モデルの訓練効率を70〜210%高速化

MITの研究チームが、大規模推論モデルの出力を予測する小型モデルを自動訓練させることで、メインモデルの計算負荷を軽減し、訓練速度を**70〜210%向上**させる手法を発表した。精度を維持しながらコストを大幅に削減できる点が評価されている。

- 出典: [MIT News](https://news.mit.edu/2026/new-method-could-increase-llm-training-efficiency-0226)

### Apple、ICLR 2026 にて MANZANO を発表

Appleは ICLR 2026 において、マルチモーダルモデル「**MANZANO**」を発表した。理解タスクと生成タスクのトレードオフを低減するよう設計された統合型モデルで、Apple Silicon（M5 Max搭載MacBook Pro）上でのローカルLLM推論も実証している。

- 出典: [Apple Machine Learning Research](https://machinelearning.apple.com/research/iclr-2026)
